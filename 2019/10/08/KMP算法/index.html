KMP算法

Posted on 2019-10-08 | In Algorithm
字符串匹配算法----KMP算法

问题描述

简单来说，就是查找模式串 P 在文本串 S 中的位置

暴力匹配算法

暴力匹配算法每次从 S 串的 i 位置开始匹配，这其中一个很大的问题是每次匹配失败后，i 都会回溯。

KMP算法

KMP算法具体流程为：
假设现在文本串 S 匹配到 i 位置，模式串 P 匹配到 j 位置

  • 如果 j = -1，或者当前字符匹配成功（即S[i] == P[j]），令 i++，j++，继续匹配下一个字符；
  • 如果 j != -1，且当前字符匹配失败（即S[i] != P[j]），则令 i 不变，j = next[j]。此举意味着失配时，模式串 P 相对于文本串 S 向右移动了 j - next [j] 位。

而 next[j] 代表 j 位置的字符与 next[j] 位置字符拥有最大长度相同前缀，其中 next[j] 位置的前缀从 S 的开头开始计算。

例如下面这个D匹配失效时，下一个 j 应该是C的位置，因为D和C拥有最长相同前缀AB

下面介绍怎么求解next数组：

利用递归求解 next 数组，已知 next[j] = k，求 next[j+1]

  • 若 p[k] == p[j]，则 next[j+1] = next[j] + 1 = k + 1
  • 若 p[k] != p[j]，则令 k = next[k]，只到 p[k] == p[j] || k == -1，此时 next[j+1] = k + 1

然后就可以写出具体算法了
